Membro:

Papel Atual:

  • Observador: o observador pode apenas visualizar o projeto, os membros e os requisitos.
  • Verificador: o verificador além de observar o projeto pode solicitar mudanças nos requisitos.
  • Gerente de Requisitos: o gerente de requisitos além de observar e verificar o projeto pode atender as solicitações e modificar os requisitos.
  • Gerente de Projetos: o gerente de projeto além de observar, verificar e gerir os requisitos do projeto pode alterar as configurações e gerenciar os membros do projeto.

Atenção! A operação excluir membro não pode ser desfeita. Caso desejado, o usuário poderá ser adicionado ao projeto novamente.
