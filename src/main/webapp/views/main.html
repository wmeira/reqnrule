Req'n'Rule

Req'n'Rule é um simples sistema para gerenciar os requisitos de seus projetos.

Criar Conta

Gerencie requisitos

Crie seu projeto e adicione seus requisitos. Gerencie o ciclo de vida de cada um deles.

Gerencie sua equipe

Adicione as partes interessadas na equipe do projeto, escolhendo o nível de permissão adequado para cada membro.

Solicitação de alterações

Os membros com permissão no projeto podem solicitar alterações de um requisito do seu projeto. Os gerentes de requisito e gerentes do projeto atendem as solicitações.
