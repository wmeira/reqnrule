Projeto
O Req'n'Rule é resultado do projeto de conclusão de curso da pós-graduação em Engenharia de Software da Universidade Federal do Paraná, turma de 2014-2015.
Descrição
Req'n'rule é um simples sistema de gerenciamento de requisitos de projeto. Nele é possível gerenciar os projetos, as partes interessadas e os requisitos do mesmo. Dependendo do nível de permissão do membro pode-se atualizar os requisitos ou enviar solicitações de mudança aos gerentes.
